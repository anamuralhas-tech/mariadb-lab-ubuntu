# mariadb-lab-ubuntu

Hands-on MariaDB lab on Ubuntu Server with SQL, user management and permissions.

## Overview
This repository documents a practical MariaDB lab built on Ubuntu Server for learning database administration, SQL fundamentals, user management and permissions.

The goal of this lab is to support hands-on practice with installation, service management, database creation, table creation, data manipulation and access control in a controlled virtual environment.

## Technologies used
- Ubuntu Server
- MariaDB
- SQL
- Apache
- PHP
- phpMyAdmin
- SSH

## Topics covered
- MariaDB installation on Ubuntu Server
- Service management with `systemctl`
- Initial hardening and security configuration
- Database creation
- Table creation
- Data insertion and queries
- SQL filtering and joins
- User creation and permission assignment
- Access verification and privilege review

## Repository structure
- `sql/` - SQL scripts used in the lab
- `docs/` - supporting notes and documentation
- `images/` - screenshots or visual evidence of the lab

## Purpose
This repository serves as a record of lab practice related to database administration in Linux environments

It is intended to demonstrate hands-on learning in MariaDB, SQL and basic access control concepts on Ubuntu Server.
